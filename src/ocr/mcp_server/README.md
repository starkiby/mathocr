# HOCR MCP Agent

### Vue Frontend

Start command:

```bash
cd hocr-vue-client/

npm install
npm run dev
```

Required packages:
```bash
cd hocr-vue-client/

npm install katex
npm install axios
```

### MCP Server

Start command:

```bash
uvicorn mcp_server:app --reload --host 0.0.0.0 --port 8000
````
